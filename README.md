# Learning Experience Design Blog

A modern Next.js blogsite about Learning Experience Design, learner-centered teaching, AI-assisted course creation, and practical instructional strategies.

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production Check

```bash
npm run lint
npm run build
```

## Deployment

Vercel builds this project with:

```bash
npm run build
```

The deployed site will update after the LXD app files are committed and pushed to the branch connected to the Vercel project.
